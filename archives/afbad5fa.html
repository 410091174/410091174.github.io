数字电路入门

0 前言

  本文主要针对缺氧游戏里的自动化，编者水平有限，基本抄书，把切合缺氧内容的知识搬出来给读者看，对数字电路感兴趣的读者可购买学习《数字电子技术基础》(清华大学)。

  本文大概讲解到读者能通过想要的结果倒推逻辑电路。

1 逻辑代数基础

1.1 二值逻辑

  我们平常所用的二进制（0和1），是像平常用十进制（0~9）一样表示数量的大小，例如手机3台(十进制) =11台(二进制)，百科群有158人(十) = 有10011110人(二)，这种使用方法本文不打算讲解，虽然数字电路有，但在缺氧里用不上，并且，编者懒。

  

  在数字逻辑电路中，用二进制数码的 0 和 1 表示一个事物的两种不同逻辑状态。

  例如，在缺氧中，

     1 表示自动化线的绿色，对建筑表示启用、工作、打开……

     0 表示自动化线的红色，对建筑表示禁用、停止、关闭……

  这种只有两种对立逻辑状态的逻辑关系称为二值逻辑。

1.2 逻辑代数的运算

1.2.1 三种基本逻辑运算

  逻辑代数的基本运算有与、或、非三种。

 1. 与

图1 与门符号
A和B是输入，Y是输出。

  只有在两个输入都收到 1 时，才输出 1 ；

  只要其中一个（或两个）输入收到 0，就输出 0。

  用逻辑表达式来表述，则可写为

L = A · B

  式中小圆点“ · ”表示 A 、B 的与运算。也称为逻辑乘。在不引起混淆的前提下，一般省略乘号“ · ”，写成 L = AB 。

表1 与逻辑真值表

 2. 或

图2 或门符号
A和B是输入，Y是输出。

  只要其中一个（或两个）输入收到 1 ，就输出 1 。

  只有在两个输入都收到 0 时，才输出 0 ；

  用逻辑表达式来表述，则可写为

L = A + B

  式中符号“ + ”表示 A 、B 的或运算，也表示逻辑加。

表2 或逻辑真值表

 3. 非

图3 非门符号
A是输入，Y是输出。

  输出与输出相反。

  当输入收到 0 时，输出 1 ；

  当输入收到 1 时，输出 0 。

  用逻辑表达式来表述，则可写为

L = A

  式中字母 A 上方的短线“ — ”表示非运算。

表3 非逻辑真值表

1.2.2 几种常用的逻辑运算

  除了三种基本运算外，还经常使用一些其他的逻辑运算，例如与非、或非、异或和同或。

  与非运算是与和非的组合运算，先进行与运算，再进行非运算。

  

逻辑表达式可写成

L = A·B

  

  或非运算是与和非的组合运算，先进行或运算，再进行非运算。

逻辑表达式可写成

L = A+B

  异或的逻辑关系是：

    当两个输入状态相同时，输出为 0 。

    当两个输入状态不同时，输出为 1 。

  用逻辑表达式来表述，则可写为

L = AB + AB = A ⊕ B

  式中符号“ ⊕ ”表示 A 、B 的异或运算。

异或逻辑真值表

  同或的逻辑关系是：

    当两个输入状态相同时，输出为 1 。

    当两个输入状态不同时，输出为 0 。
  用逻辑表达式来表述，则可写为

L = AB + AB = A ⊙ B

    真值表略（懒）

  同或与异或的逻辑关系正好相反，相当于异或门的输出端串联一个非门。
