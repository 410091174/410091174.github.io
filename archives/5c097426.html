游戏操作

1 建造

  建造分3个阶段：

  （1）安放蓝图——玩家选择建筑和建造地点。

  （2）复制人运送材料给蓝图。

  （3）复制人建造建筑。
  如果复制人无法到达建造地点，则该建筑的提示信息中显示“无法到达的建造地点”。
  资源不足有“资源不足”图标提示。

2 操作

挖掘（快捷键：G）

给复制人安排挖掘或拔除植物任务。(确定不是用激光枪突突突？)

掉落物俗称“碎片”，并且挖掘掉落物的质量只有原来固体块的一半。
坚硬的固体块需要复制人学习一定等级的挖掘技能才能被挖掘，挖掘图标中“菱形图案”的个数代表所需的挖掘等级，如花岗岩1级、深渊晶石2级、黑曜石3级……
中子物质能被有3级挖掘技能的复制人挖掘，但挖掘进度不会增加，无法被成功挖掘。（这小人挖足了9981天啊）

拆除（快捷键：X）

给复制人安排拆除任务。 （1）拆除建筑后如数掉落原材料，建造时花费多少就掉落多少。 （2）有少量不能拆除的建筑，如打印舱、遗迹中的衣柜、电脑等。

拆除命令是默认拆除框选中的全部建筑，如果多个建筑叠加在一格，要拆除某一建筑的同时又不影响其他建筑，这时有两种方法： （1）点击拆除命令后，选定“工具过滤器”中要被拆除的建筑种类。 （2）先进入相应的视图再点击拆除命令，工具过滤器会自动锁定到相应的建筑种类。(此方法拆电线、管道、自动化线更方便）
举例1：砖块、电线叠在一格，要求拆除砖块但不影响电线和管道。
如果直接用拆除命令去框选，那么砖块和电线都会被标上“拆除”，这明显是不符合要求的，那么应该怎么操作才能只拆掉砖块呢？
在工具过滤器中选定建筑种类，此处要求是拆除砖块，故切换至建筑。
再进行框选，可以看到，只有砖块有“拆除任务”。
举例2：拆除电线且不影响其他建筑。
除了能在工具过滤器中选定建筑种类，还可以先进入“电力视图”，再用拆除命令，工具过滤器会自动锁定到“电线”，一样能达到相同的效果，并且还方便玩家看电线布局。

取消（快捷键：C）

取消先前下达的操作。

可以通过“工具过滤器”或“进入相应的视图”取消对应的操作。

优先度（快捷键：P）

对“任务”进行优先度排序，优先度高的先做，优先度低的后做。 相当于让复制人先做什么再做什么。

优先度默认为5， 1->9 = 无关紧要->非常重要。

感叹号为紧急，使用后会先打断所有复制人任务，最先被打断的复制人会去完成感叹号任务。（编者也不清楚谁会最先被打断，希望知道的读者能在此留下批注或在反馈贴留言。）

操作命令、建造、绝大部分机器……都有优先度，合理安排优先度能让蠢蠢的复制人分清任务轻重从而井然有序地工作，对整体的工作效率有极大提高。

清扫

给复制人安排清扫杂物任务。

“清扫”的描述是“把杂物移到储物箱中”，其实这句话是不全面的。

清扫的正确描述是：

优先使用被清扫的杂物。（杂物：碎片、瓶装液体/气体）

把杂物运到储物箱是使用杂物的一种方式，运到等待建造的建筑也是，供应给机器也是……

其实就算不清扫，复制人也会自发地把杂物运到储物箱/空桶器/空罐器中，只是被框上清扫的杂物会被优先运送。

储物箱满了或储物箱/空瓶器/空罐器没有勾上相应的物质，杂物的清扫图标会变成红色，同时提示信息显示“储物箱不可用”，即杂物无法被清扫进储物箱/空瓶器/空罐器，但还能被优先使用。

仅限清扫：储物箱/空瓶器/空罐器勾上“仅限清扫”后，复制人不再自发把杂物运到该储物箱/空瓶器/空罐器，需要玩家对杂物框上清扫后复制人才会把杂物运过来。

擦拭

给复制人安排擦拭液体任务。 擦拭是把液体擦成瓶装液体。

擦拭只能标记在地面上并且是150kg/格及以下的液体。
以下是不能标记擦拭的情况：
液体过多，大于150kg/格。 如果标记时是小于150kg/格，标记后该格液体增加到150kg以上，复制人也能擦拭。
非地板，即液体不在地面上。

收获

点击“收获命令“后进入收获视图，如右图。 上方是种植的植物，默认是“绿色镰刀标志” 下方是野生植物，默认是“红色禁止标志”。 （1）绿色镰刀标志的植物成熟后复制人会主动收获。 （2）红色禁止标志的植物复制人不会去收获。
启用收获：把“红色禁止标志”变成“绿色镰刀标志”。 禁用收获：把“绿色镰刀标志”变成“红色禁止标志”。

消毒

给复制人安排消毒任务。

杂谈：

云中影（百科群群友）：“不建议给厕所消毒，这东西永远带菌，消毒纯粹浪费时间”

编者：

的确，对于缺氧来说，消毒是没必要的，建议把消毒关掉。

在这说一下”食物中毒菌”的源头：小人的尿。

一般来说，食物中毒菌传播开来，也就下面几种情况：

1.上厕所没洗手，小人上完厕所之后身上会带菌，出来洗个手就能洗掉全部细菌，所以如果家里除了厕所以外有食物中毒细菌，可以看看是不是小人没洗手（厕所：洗手盆=1：1）。

2.没得上厕所，随地尿尿，这种情况五花八门，自己想办法遏制。

可见，解决方法就是厕所问题，搞好厕所相应设施之后，食物中毒菌只会在厕所积累，但传播不出去，也就没必要对厕所进行消毒了。

攻击动物

给复制人安排攻击（杀死）动物任务。 可怜的小动物 T﹏T 好吃的烤肉串 (╹ڡ╹)

捕捉动物

给复制人安排捕捉动物任务。 复制人执行该任务需要“放牧技能”。

把动物移动到其他地方需要用“小动物放生点”，在小动物放生点的信息窗口勾选上相应的动物后，复制人会把动物运到放生点处。（麻袋套动物.gif)

取出管道内容物

给复制人安排取出管道内容物任务。 (1)复制人执行该任务需要“管道工作”技能。 (2)把管道内容物变成瓶装液体/气体，轨道内容物变成碎片。

以液体管道举例：

标记上管道清空（管道液体不会停止流动），被清出来的是读条完成时刚好在被标记管道上的那节液体。

需要注意的是，复制人会一直清空被标记的管道，直到该节管道无液体为止。若液体一直流动至该节管道，则复制人一直清理。

3 常用快捷键

旋转建筑（快捷键：O，英文字母）

对建筑进行90°或180°旋转。建造时有“按[O]来旋转”的建筑都能旋转，没有这句话的建筑不能旋转。

把镜头移至打印舱处（快捷键：H）

把镜头移至打印舱处，并且会缩小视野，效果如下图。

放大视野范围（快捷键：Alt + S）

按下 Alt + S 后滚动鼠轮能获得更大的视野范围，再次按下 Alt + S 或 Esc
可回到正常界面。不过 Alt + S 会隐藏游戏界面(UI)。

标记地点和移动镜头至标记地点

快捷键：

标记地点 Ctrl + 1 Ctrl + 2 Ctrl + 3 …… Ctrl + 0
移动镜头至标记地点 Shift + 1 Shift + 2 Shift + 3 …… Shift + 0

用 Ctrl + 1 标记地点后，按下 Shift + 1 快速把镜头移动到该地点。

标记时还会记录下当时的视野大小，按下移动镜头的快捷键后会把视野大小调整成标记时的视野大小。
